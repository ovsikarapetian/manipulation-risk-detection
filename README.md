# Manipulation Risk Detection System
Yerevan State University — Thesis 2026
Ovsanna Karapetyan

## Description
Vocal emotion-based manipulation risk assessment system.
MFCC features + SVM model + Rule-based risk module.
Trained on RAVDESS dataset — 5 emotions.
Accuracy: 90.36%

## Installation
pip install librosa scikit-learn numpy joblib

## Usage
python main_fixed.py
python predict_enhanced_safe.py

## Emotions
- Neutral
- Happy
- Sad
- Angry
- Fear
